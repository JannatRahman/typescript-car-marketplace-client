# 🚀 Installation

## Clone Client

```bash
git clone https://github.com/your-username/typescript-car-marketplace-client.git
```

Install dependencies

```bash
npm install
```

Run

```bash
npm run dev
```

---

## Clone Server

```bash
git clone https://github.com/your-username/typescript-car-marketplace-server.git
```

Install dependencies

```bash

```

Run

```bash
npm run dev
```

# 🔐 Environment Variables

## Server

Create a `.env` file.

```env
PORT=5000

DB_USER=your_mongodb_username
DB_PASS=your_mongodb_password

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:3000
```

---

## Client

Create `.env.local`

```env
NEXT_PUBLIC_API_URL=http://localhost:5000
```
